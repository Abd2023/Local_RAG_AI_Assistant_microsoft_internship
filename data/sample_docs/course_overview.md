# Northstar AI Summer School Course Overview

Northstar AI Summer School is a fictional four-week program where beginner computer science students build a local document question-answering assistant. The course focuses on Retrieval-Augmented Generation, local model inference, embeddings, SQLite storage, and responsible answer behavior.

The main project is a local RAG assistant that answers questions from a small handbook dataset. Students begin with a command-line interface before adding any optional web interface. The first goal is to understand every pipeline step rather than hide the work inside a framework.

The program meets Monday through Friday from 10:00 AM to 4:00 PM. The morning block is used for lessons, guided examples, and short quizzes. The afternoon block is reserved for coding practice, debugging, and project checkpoints.

Students work in teams of two or three. Each team must keep a shared README that explains the purpose of the assistant, how to run it, what documents it uses, and what limitations remain. Teams may use public documentation and instructor-provided examples, but they must write and understand their own project code.

By the end of the program, every team should demonstrate an offline assistant that can retrieve relevant chunks, build a grounded prompt, and answer using a local language model. The assistant should also say that it does not know when the answer is not available in the provided documents.
